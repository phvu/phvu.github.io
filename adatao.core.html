cut(column, break_type='equalInterval', breaks=2, include_lowest=False, right=True, digit_label=2)[source]¶

Convert a DistributedDataFrame numeric column to an DistributedFactor one. Return the current DDF if it is mutable, otherwise a new DDF.

This function divides the range of a numeric column into intervals and codes the values according to which interval they fall. The leftmost interval corresponds to level one, the next leftmost to level two and so on. :param column: column name or index :param break_type: ‘custom’, ‘equalInterval’ or ‘equalFreq’

‘custom’: custom intervals - users supply an array of break points. ‘equalInterval’: intervals of equal size. ‘equalFreq’: intervals of equal number of values.
Parameters:
  • breaks – either a numeric vector of two or more unique cut points or a single number (greater than or equal to 2) giving the number of intervals into which column is to be cut.
  • include_lowest – logical, indicating if an ‘x[i]’ equal to the lowest (or highest, for right = FALSE) ‘breaks’ value should be included.
  • right – logical, indicating if the intervals should be closed on the right (and open on the left) or vice versa.
  • digit_label – integer which is used when labels are not given. It determines the number of digits used in formatting the break numbers.
Returns:

A new DistributedDataFrame with the same properties as x, except the ‘column’ which is now of DistributedFactor category. Values which fall outside the range of breaks are coded as NA, as are NaN and NA values. Return the current DDF if it is mutable, otherwise a new DDF
drop_na(margin='row', how='any', thresh=0)[source]¶

Drop rows/columns with missing values :param margin: ‘row’ or ‘column’ :param how: ‘all’ or ‘any’, controlling in what condition a row/column is dropped. If how = ‘any’, a row/column will be dropped if it contains at least one missing value. If how = ‘all’, a row/column will only be dropped if all of its value are missing. This parameter will be overridden by {@code thresh} parameter if {@code thresh} is set. :param thresh: minimum number of missing values to trigger a drop action. :return: a DistributedDataFrame with rows or columns with missing values being dropped if the current DDF is immutable. Otherwise, the rows or columns are dropped in place.
