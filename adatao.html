Module contents¶

pAnalytics’ Python interface

This package implements Python interface for pAnalytics. Typically it works like this:

>>> s = adatao.connect(host, server_ip, username, password)

s is a adatao.Session object, representing a working session to the server.

On s, you can run several functions:

>>> s.list_ddfs()
>>> s.list_ddf_metadata()

You can get a Distributed DataFrame by its URI:

>>> df = s.get_ddf('ddf://adatao/<ddf_name>')

df is a adatao.DistributedDataFrame object that represents a Distributed DataFrame on pAnalytics server. On df, you can run various functions:

>>> df.summary()
>>> df.five_nums()

Check https://github.com/adatao/PyClient for more information.
